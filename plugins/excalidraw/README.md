# Excalidraw 绘图工具插件

基于 Excalidraw 的离线绘图工具，支持手绘风格的图表和白板功能。

## 功能特性

- ✨ 手绘风格的绘图体验
- 🎨 丰富的绘图工具（矩形、圆形、箭头、文字等）
- 📁 本地文件保存和加载
- 🎯 支持多种导出格式（PNG、SVG、JSON）
- 🌍 中文界面支持
- 📱 响应式设计，适配不同屏幕尺寸

## 使用方法

1. 在 UniHub 插件市场中安装 Excalidraw 插件
2. 启动插件后即可开始绘图
3. 使用工具栏中的各种工具进行创作
4. 通过主菜单保存或加载文件

## 快捷键

- `Ctrl/Cmd + Z`: 撤销
- `Ctrl/Cmd + Y`: 重做
- `Ctrl/Cmd + C`: 复制
- `Ctrl/Cmd + V`: 粘贴
- `Delete`: 删除选中元素
- `Ctrl/Cmd + A`: 全选

## 技术栈

- React 18
- TypeScript
- Vite
- @excalidraw/excalidraw

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包插件
npm run package
```
